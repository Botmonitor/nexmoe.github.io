Typecho 主题 Nexmoe

2019年08月25日 305 字 大概 1 分钟

一个比较特别的 Typecho 主题

Github

https://github.com/nexmoe/typecho-theme-nexmoe
欢迎 Star，贡献，如遇到问题请留言





获取更新

使用 watch Releases only 以获取新版本更新提示

开源协议

Apache License 2.0

后续更新

就读高中，
star 以及 issue 是我更新的动力
欢迎加群 482634342 划水

特性

  • 相片集
  • 图片灯箱
  • 文章封面图

国际化

目前中文翻译较全，其他语言翻译不完全，有余力的大佬可以来贡献一下
语言文件在 languages 里，参考 zh-CN.yml 进行翻译即可
